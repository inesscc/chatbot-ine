    Uso diario¶

    Una vez que hiciste la instalación inicial, el uso diario es muy simple.

    Iniciar el proyecto¶

    # 1. Abre la Terminal

# 2. Ve a la carpeta del proyecto
cd /Users/tu_nombre/proyectos/chatbot/open-webui

# 3. Inicia el proyecto
make prod-up

    # 1. Abre PowerShell

# 2. Ve a la carpeta del proyecto
cd C:\Users\tu_nombre\proyectos\chatbot\open-webui

# 3. Inicia el proyecto
docker compose --profile prod up -d

    Espera ~1 minuto y abre http://localhost:3030 en tu navegador.

    Detener el proyecto¶

    Cuando termines de trabajar, apaga los contenedores para liberar recursos. Los datos se guardan automáticamente.

    make prod-down

    docker compose --profile prod down

    Referencia rápida de comandos¶

    Acción macOS Windows
    Iniciar make prod-up docker compose --profile prod up -d
    Detener make prod-down docker compose --profile prod down
    Ver estado make status docker ps
    Ver logs en vivo make prod-logs docker compose --profile prod logs -f
    Reiniciar make prod-restart docker compose --profile prod down && docker compose --profile prod up -d

    Detener los logs sin apagar el proyecto

    Cuando ves los logs con make prod-logs (o docker compose ... logs -f), puedes detener la visualización con Ctrl + C en cualquier momento. Esto no apaga los contenedores, solo deja de mostrar los mensajes en pantalla.
